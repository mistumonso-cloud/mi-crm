Badge semántico genérico — el color comunica estado, nunca decora.

```jsx
<Badge tone="success" dot>Activo</Badge>
<Badge tone="warning">Vence hoy</Badge>
```

Tonos: `success` (verde), `warning` (ámbar), `danger` (rojo), `neutral` (gris), `accent` (azul pizarra). Para estados del pipeline de ventas usa `StatusBadge` en su lugar.
